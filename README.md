# OfferAgent · 职说

**老H · 你的求职陪练** — 面向大专应届生的求职全流程辅助工具。

## 功能

- **简历诊断** — 上传简历（PDF/Word/TXT），逐项评分 + 改写建议
- **找机会** — 招聘渠道对比、投递黄金时间、打招呼 & 邮件模板、避坑清单
- **接通知** — 面试电话 9 步法
- **备面试** — 自我介绍分析、穿搭速查
- **过面试** — 面试题库（陷阱题 / 压力题 / STAR 追问），举止红黑榜
- **谈薪资** — 城市 × 行业 × 学历薪资测算 + 谈判话术
- **随时问** — 右下角悬浮聊天窗，AI 陪练随时答疑

## 技术栈

纯前端单页，基于 Coze API v3 提供 AI 能力，Token 保存在浏览器 localStorage 中。

## 使用方式

直接用浏览器打开 `index.html`，点击右上角设置 Coze Token 后即可使用全部功能。
